# rick-asli
CS 370 Term Project: Rick ASLI (aka Richard American Sign Language Interpreter)

basic idea:

- camera takes picture (on an interval)
- picture saved to rpi file system
- rpi processes picture via gaussian edge detection (probly?)
- rpi finds closest match to library of all other letters
- rpi writes to client pc the matching letter
- client pc strings together letters into words (yay)
